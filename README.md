# FZM Website

A website featuring project catalog, estimators, and contact information.

## Features

- **Home Page**: Introduction and overview
- **Projects**: Showcase of completed projects
- **Catalog**: Product/service catalog
- **Estimators**: Interactive cost estimators powered by Google Sheets
- **Contact**: Contact information and form

## Structure

- `index.html` - Home page
- `projects.html` - Projects showcase
- `catalog.html` - Product catalog
- `estimators.html` - Interactive estimators
- `contact.html` - Contact page
- `assets/` - Static assets (CSS, JavaScript, images)

## Estimators

The estimators feature pulls questions dynamically from a Google Sheet, allowing for easy updates without code changes.

## Usage

Simply open `index.html` in a web browser to view the site locally.
